# Pixel Snake Game

A classic Snake game built with pure HTML, CSS, and Vanilla JavaScript, utilizing the HTML5 Canvas API.

## Features

- **Classic Gameplay**: Eat food to grow longer, and avoid hitting the walls or yourself.
- **Configurable Settings**:
  - Adjust snake speed (Slow, Medium, Fast).
  - Toggle whether the snake is allowed to pass through walls.
- **High Score Tracking**: Keeps track of your highest score using browser `localStorage`.
- **Pause/Resume functionality**: Pause the game at any time during gameplay.

## Technologies

- HTML5 Canvas for game rendering
- Vanilla JavaScript for game logic
- CSS for layout and styling
- ESLint and Prettier for code formatting and quality
